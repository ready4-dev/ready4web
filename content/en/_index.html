ready4 aims to be accountable, flexible and up-to-date in modelling systems that shape the mental health and wellbeing of young people.

{{% /blocks/lead %}} {{< blocks/section color="light">}} {{< cardpane >}} {{< card header="People" >}} Currently available modules and unreleased work-in-progress modules for modelling characteristics, relationships, behaviours, risk factors and outcomes. {{< /card >}} {{< card header="Places" >}} A currently available app and unreleased work-in-progress modules for modelling demographic, environmental and proximity drivers of access, equity and outcomes. {{< /card >}} {{< /cardpane >}} {{< cardpane >}} {{< card header="Platforms" >}} Unreleased work-in-progress modules for modelling the optimal staffing and configuration of support services. {{< /card >}} {{< card header="Programs" >}} Highly preliminary modules for modelling the affordability, value for money and appropriate targeting of interventions. {{< /card >}} {{< /cardpane >}} {{< /blocks/section >}} {{< blocks/lead >}}

ready4 is being developed as multiple independent modelling projects, that adopt a common framework to iterate over three steps.

{{< /blocks/lead >}} {{< blocks/section color="dark">}} {{% blocks/feature icon="fab fa-connectdevelop" title="[1. Develop model](/docs/model/)" %}} Abstract and generalised data structures and algorithms are written in computer code. {{% /blocks/feature %}} {{% blocks/feature icon="fas fa-database" title="[2. Add Data](/docs/datasets)" %}} Data specific to a decision context and set of assumptions are added to data structures. {{% /blocks/feature %}} {{% blocks/feature icon="fas fa-chart-line" title="[3. Run Analyses](/docs/analyses)" %}} Algorithms are applied to data contained in data structures to generate insight. {{% /blocks/feature %}} {{< /blocks/section >}} {{< blocks/lead>}}

ready4 has been developed for three main types of user and contributor.

{{< /blocks/lead >}} {{< blocks/section color="dark">}} {{% blocks/feature icon="fas fa-chalkboard-teacher" title="[Coders](/docs/getting-started/users/coder/)" %}} [Contribute](/docs/contribution-guidelines/contribution-types/code/) to [software](/docs/getting-started/software/) with a social purpose - help [author](/docs/model/authoring-modules/) ready4 [modules](/docs/getting-started/concepts/module/). {{% /blocks/feature %}} {{% blocks/feature icon="fas fa-user-cog" title="[Modellers](/docs/getting-started/users/modeller/)" %}} [Use ready4 modules](/docs/model/using-modules/) to undertake novel modelling analyses and to [author and share model datasets](/docs/datasets/authoring-data/). {{% /blocks/feature %}} {{% blocks/feature icon="fas fa-handshake" title="[Planners](/docs/getting-started/users/planner/)" %}} Co-develop, commission and use [decision-aids](/docs/analyses/decision-aids/) built from ready4 modules and data. {{% /blocks/feature %}} {{< /blocks/section >}} {{< blocks/lead >}}
