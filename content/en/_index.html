ready4 adopts a collaborative and iterative approach to modelling systems that shape the mental health and wellbeing of young people.

{{% /blocks/lead %}} {{< blocks/section color="light">}} {{< cardpane >}} {{< card header="People" >}} Modelling the characteristics, relationships, behaviours, risk factors and outcomes of young people and those important to them. {{< /card >}} {{< card header="Places" >}} Modelling demographic, environmental and proximity drivers of access, equity and outcomes. {{< /card >}} {{< /cardpane >}} {{< cardpane >}} {{< card header="Platforms" >}} Modelling the optimal staffing and configuration of support services for young people. {{< /card >}} {{< card header="Programs" >}} Modelling the affordability, value for money and appropriate targeting of interventions for young people. {{< /card >}} {{< /cardpane >}} {{< /blocks/section >}} {{< blocks/lead >}}

ready4 is being developed as multiple independent modelling projects, that adopt a common framework to iterate over three steps.

{{< /blocks/lead >}} {{< blocks/section color="dark">}} {{% blocks/feature icon="fab fa-connectdevelop" title="[Develop model](/docs/model/)" %}} Abstract and generalised data structures and algorithms are written in computer code. {{% /blocks/feature %}} {{% blocks/feature icon="fas fa-database" title="[Add Data](/docs/datasets)" %}} Data specific to a decision context and set of assumptions are added to data structures. {{% /blocks/feature %}} {{% blocks/feature icon="fas fa-chart-line" title="[Run Analyses](/docs/analyses)" %}} Algorithms are applied to data contained in data structures to generate insight. {{% /blocks/feature %}} {{< /blocks/section >}} {{< blocks/lead>}}

ready4 has been developed for three main types of user.

{{< /blocks/lead >}} {{< blocks/section color="dark">}} {{% blocks/feature icon="fas fa-chalkboard-teacher" title="[Coders](/docs/getting-started/users/coder/)" %}} [Test, refine and extend the software that implements ready4](/docs/contribution-guidelines/contribution-types/code/). {{% /blocks/feature %}} {{% blocks/feature icon="fas fa-user-cog" title="[Modellers](/docs/getting-started/users/modeller/)" %}} [Apply ready4](/docs/contribution-guidelines/contribution-types/use/) to undertake replicable modelling analyses. {{% /blocks/feature %}} {{% blocks/feature icon="fas fa-handshake" title="[Planners](/docs/getting-started/users/planner/)" %}} Co-develop, commission and use [ready4 decision-aids](/docs/analyses/decision-aids/). {{% /blocks/feature %}} {{< /blocks/section >}} {{< blocks/lead >}}
